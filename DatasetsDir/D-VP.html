Description

West-Nile Virus Prediction (D-VP and D-VP-r): In this competition, the participants are given weather, location, testing, and spraying data to predict whether or not West NileVirus is present. We use the winning code from Cardal, which added three new featuresto the raw dataset. The predictions are initialized according to a normal distribution. Eachprediction is then multiplied by various coefficients obtained from other information relatedto the target (e.g., geographical information). The predictions are then normalized. We abbreviate this dataset as D-VP.

The column name of the Label is: WnvPresent The features are stored in: VirusPrediction_train.csv VirusPrediction_test.csv The id of test data is stored in: id.csv

Azure experiments are available for download.

Scripts for inferece on SageMaker are available for download.
